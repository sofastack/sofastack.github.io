用户案例

编辑
更新时间: 2024-06-18
  1. RheaKV：基于 JRaft 和 RocksDB 实现的嵌入式、分布式、高可用、强一致的 KV 存储类库。
  2. AntQ Streams QCoordinator： 使用 JRaft 在 coordinator 集群内做选举、元信息存储等功能。
  3. SOFA 服务注册中心元信息管理模块：IP 数据信息注册，要求写数据达到各个节点一致，并且在不小于一半节点挂掉，保证不影响数据正常存储。
